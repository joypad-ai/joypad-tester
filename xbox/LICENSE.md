(clang + lld + pbkit + SDL2) and links against libusbohci's USB host
class drivers (HID / MSC / UAC) for keyboard / mouse / Memory Unit /
Voice Communicator detection. Those upstream components keep their own
licenses; the Joypad Tester scaffolding here is licensed under the MIT
